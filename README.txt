Суть такая. Сделать некую систему для управления заездами.
Идея такая:
Есть несколько гоночных команд.(одна сущность). Есть гонщики команд.(каждый гонщик – тоже некая сущность). Условимся что в каждой команде по 3 гонщика.
Плюс так же у команд может быть несколько машин(не ограничиваем их количество).
Каждая машина  - тоже некая сущность(Машины пусть описываются некими свойствами как мощность, Марка, модель.). 
Предположим на автодроме есть несколько треков. И одновременно может проходить сразу несколько гонок.
Каждая команда может выставлять своих участников для гонки на каком то треке. 

Посему нужно создать сущность Заезд. Заезд может находится в трех состоянияx. Это CREATED(все участники добавлены. Все на старте)
ONGOING- заезд в процессе, гонщики катаются по трассе. И FINISHED – заезд окончен. 
Тут уже должны быть определен победитель. Будем как американцы, потчевать только первого. Поэтому победитель один.
Также нужно хранить информацию о всех гонщиках которые учавствовали в этом заезде и на каких машинах.

Доп условия:
Команда может выставить одного гонщика для гонки на нескольких треках. Но естественно одновременно он там и там быть не может.
Поэтому нужно сделать так что если гонщик уже учавствует в каком то заезде – то второй заезд в котором этот гонщик зарегистрирован – начаться не может. Нужно дождаться окончания первого заезда.

Технические детали:
Взаимодействие с бэкэндом просиходит через REST. По сути старт заезда – это PUT запрос на апдейт статуса заезда.
Соотвественно нужно создать CRUD операции для всех сущностей как гонщик, команда, заезд, машина.

Вся информация о машинах, гонщиках , заездах должна храниться в базе данных.

Должна быть реализована возможность получать информацию о всех гонщиках, машинах, заездах.
Для гонщиков реализовать возможность фильтрации по командам  и к примеру по количеству побед (победа для упращения это у нас значит что он занял первое место в каком то заезде.
Т.е. побед может быть от 0 и до 100500…)
Для заездов нужна возможность фильтрации по гонщику. Т.е. должна быть возможность показать все заезды где принимал участие определнный гонщик. 
И так же сделать фильтрацию по автомобилям. Т.е. показывать заезды для определенного гонщика на определенной машине. Ну и просмотр вообще всех заездов тоже должен быть.
